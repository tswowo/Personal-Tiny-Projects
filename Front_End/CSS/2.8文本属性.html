文本属性
  1. text-align 指定元素文本的对齐方式
    值 描述
    left 文本居左排列(默认值)
    right 文本居右
    center 文本居中
  2. text-decoration 指定元素文本的修饰，下划线、上划线、删除线等
    值 描述
    underline 定义下划线
    overline 定义上划线
    line-through 定义删除线
  3. text-transform 控制文本的大小写
    值 描述
    captialize 定义每个单词开头大写
    upperCASE 定义全部大写字母
    lowerCASE 定义全部小写字母
  4. text-indent 控制文本块中的首行缩进
    值 描述
    0px 缩进
    30px 首行向右缩进
    30px
    -30pxE 首行向左缩进
    30px
