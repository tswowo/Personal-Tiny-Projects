背景属性
其实元素的背景大小是元素的总大小，包括填充和边界。通常在左上角，且通常如果不够大则会向右、下重复填充
CSS的背景属性主要内容如下：
属性 描述
background-color 设置背景颜色
background-image 设置背景图片
background-repeat 设置背景图片如何填充
background-size 设置背景图片大小属性
background-position 设置背景图片显示位置

    repeat-y 只垂直平铺
    no-repeat 不平铺
  4. 我是
    background-size
    背景图片大小示例
    值 说明
    length 两个值，分别为宽度和高度，若只设定一个，那么另一个值auto
    percentage 相对区域百分比，仍然会进行auto处理
    cover 保持纵横比并将图片缩放成完全覆盖背景区域的最大大小
    contain 保持纵横比并将图片缩放成完全覆盖背景区域的最小大小
  5. 我是图片位置示例
    值 说明
    水平位置(left right center)+垂直位置(top bottom center) 如left center为左 中
    x% y% 左上角为0%0% 右下角为100%100%
    指定一个值时，另一个值为50%
    默认全为0%0%
    xpos ypos 单位为像素
